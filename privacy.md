# Política de Privacidade — Ricciardo Bot

Última atualização: 19 de junho de 2026

Esta Política de Privacidade explica quais informações o Ricciardo Bot coleta, como essas informações são usadas e como são protegidas.

Ao utilizar o Ricciardo Bot, você concorda com esta Política de Privacidade.

## 1. Informações coletadas

O Ricciardo Bot pode armazenar algumas informações necessárias para o funcionamento dos comandos e sistemas do bot, incluindo:

* ID do usuário do Discord;
* ID do servidor;
* ID de canais utilizados em comandos específicos;
* cards obtidos pelo usuário;
* moedas virtuais;
* packs comprados ou usados;
* histórico necessário para inventário, drops, trades e transferências;
* configurações de lembretes, quando ativadas pelo usuário;
* dados técnicos necessários para funcionamento do bot.

O bot não coleta nome real, endereço, telefone, documentos pessoais, senhas, dados bancários ou informações sensíveis.

## 2. Como os dados são usados

As informações coletadas são usadas apenas para permitir o funcionamento do Ricciardo Bot, incluindo:

* registrar inventários de cards;
* controlar moedas virtuais;
* permitir drops e cooldowns;
* gerenciar packs, loja e compras virtuais;
* realizar trades e transferências;
* ativar lembretes solicitados pelo usuário;
* corrigir bugs, duplicações ou erros no sistema;
* manter a segurança e integridade da economia do bot.

## 3. Armazenamento dos dados

Os dados do bot podem ser armazenados em banco de dados local ou outro sistema utilizado pela equipe responsável pelo Ricciardo Bot.

Esses dados são mantidos apenas pelo tempo necessário para o funcionamento do bot, manutenção da economia, suporte aos usuários e prevenção de abuso.

## 4. Compartilhamento de dados

O Ricciardo Bot não vende, aluga ou compartilha dados dos usuários com anunciantes ou terceiros para fins comerciais.

Os dados podem ser acessados apenas pela equipe responsável pelo bot quando necessário para manutenção, suporte, investigação de bugs, prevenção de abuso ou cumprimento de obrigações técnicas e legais.

## 5. Dados visíveis para outros usuários

Algumas informações do bot podem ser exibidas publicamente dentro do Discord, dependendo do comando utilizado, como:

* cards obtidos em drops;
* inventário, quando exibido pelo usuário;
* resultados de trades;
* transferências confirmadas;
* menções de usuários;
* rankings ou listagens, caso existam.

Ao usar comandos públicos, o usuário entende que determinadas informações do jogo podem ser vistas por outras pessoas no servidor.

## 6. Lembretes e notificações

Caso o usuário ative lembretes, o bot pode armazenar o ID do usuário e o ID do canal para enviar mensagens quando comandos como drop ou work estiverem disponíveis.

O usuário pode desativar esses lembretes usando o comando apropriado, como `/lembrar off`, quando disponível.

## 7. Segurança

A equipe do Ricciardo Bot busca manter os dados protegidos e utiliza as informações apenas para o funcionamento do bot. No entanto, nenhum sistema é totalmente livre de riscos.

Em caso de falha, bug ou problema técnico, a equipe poderá realizar ajustes, correções ou limpezas no banco de dados para preservar a integridade do sistema.

## 8. Remoção de dados

Usuários podem solicitar a remoção de dados relacionados ao uso do Ricciardo Bot, quando tecnicamente possível. A remoção pode afetar inventário, moedas, cards, packs e progresso dentro do bot.

Para solicitar remoção de dados, entre em contato com a equipe responsável pelo bot pelo canal de suporte ou servidor oficial.

## 9. Uso por menores de idade

O Ricciardo Bot é destinado ao uso dentro do Discord e deve ser utilizado de acordo com os Termos de Serviço do Discord. Usuários menores de idade devem seguir as regras da plataforma e as orientações de seus responsáveis legais.

## 10. Alterações nesta Política

Esta Política de Privacidade pode ser atualizada a qualquer momento para refletir mudanças no bot, novas funcionalidades ou ajustes legais e técnicos.

O uso contínuo do bot após alterações significa que você aceita a versão mais recente desta Política.

## 11. Contato

Em caso de dúvidas sobre privacidade, dados armazenados, remoção de informações ou funcionamento do Ricciardo Bot, entre em contato com a equipe responsável pelo bot pelo servidor oficial, canal de suporte ou outro meio informado pela administração.
